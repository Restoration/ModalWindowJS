Simple ModalWindow

jQuery ModalWindow

It is a modal window simple plugin.
If you want to see more than one
Since there are a large number of plug-ins that use the value of the name and href, in order to avoid duplication with other plug-ins
Get the value from the data attribute, to display the class take that name.

JavaScript

				$(function(){
					//Default options
					$(document).modalWindow({
						"openTrigger": ".trigger",//Classes related to the open trigger
						"closeTrigger": ".closeBtn",//Classes related to the close button
						"modalContent": ".modalContent",//Classes related to the contents of the modal window
						"overLay" : "overLay",//overlayer class
						"width" : 500,//ModalWindow width
						"height": 500,//ModalWindow height
						"feadSpeed" : 500,//ModalWindow feed speed
					});
				});
